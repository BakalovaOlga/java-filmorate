Описание таблиц: <br/>
<h3>1. users</h3> 
Хранит информацию о пользователях:
<ul>
<li>user_id - уникальный идентификатор пользователя (первичный ключ).</li>
<li>email - электронная почта пользователя.</li>
<li>login - логин пользователя.</li>
<li>user_name - имя пользователя (может быть пустым).</li>
<li>birthday - дата рождения пользователя.</li>
</ul> <br/>

<h3>2. friends</h3>
Хранит информацию о дружбе между пользователями:
<ul>
<li>user_id и friend_id - идентификаторы пользователей, которые являются друзьями.</li>
</ul> <br/>

<h3>3. films</h3>
Хранит информацию о фильмах:
<ul>
<li>film_id - уникальный идентификатор фильма (первичный ключ).</li>
<li>film_name - название фильма.</li>
<li>description - описание фильма.</li>
<li>release_date - дата выхода фильма.</li>
<li>duration - продолжительность фильма в минутах.</li>
<li>rating_id - идентификатор рейтинга MPA (внешний ключ к таблице Mpa).</li>
</ul> <br/>

<h3>4. Mpa</h3>
Хранит информацию о рейтингах MPA:
<ul>
<li>rating_id - уникальный идентификатор рейтинга (первичный ключ).</li>
<li>rating_name - название рейтинга ("G", "PG", "PG-13", "R", "NC-17").</li>
</ul> <br/>

<h3>5. film_genre</h3>
Связывает фильмы с жанрами:
<ul>
<li>film_id - идентификатор фильма (внешний ключ к таблице film).</li>
<li>genre_id - идентификатор жанра (внешний ключ к таблице genre).</li>
</ul> <br/>

<h3>6. genre</h3>
Связывает фильмы с жанрами:
<ul>
<li>genre_id - уникальный идентификатор жанра (первичный ключ).</li>
<li>genre_name - название жанра.</li>
</ul> <br/>

<h3>7. likes</h3>
Хранит информацию о лайках, которые пользователи ставят фильмам:
<ul>
<li>user_id - идентификатор пользователя (внешний ключ к таблице users).</li>
<li>film_id - идентификатор фильма (внешний ключ к таблице films).</li>
</ul> <br/>
